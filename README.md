(卡1, 卡2) 600,2,3                   （一共600元  一共两种地区， 3笔交易 ）作为边的属性 
(卡2, 卡3) 300,2,2                   一共300元  一共两种地区， 2笔交易

思路，对于同一个键值对(卡1, 卡2)    弄一个临时的set，对于某个地区，如果已经出现在set中的 地区值为0， 对于没出现在set中的地区值为1， m对于每笔交易交易次数都设为1
然后使用reduceByKey即可。
